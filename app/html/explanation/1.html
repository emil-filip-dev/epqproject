Explanation Page 1: Building blocks

All matter is made up of tiny structures called atoms.

Atoms are made of a nucleus (containing protons and neutrons) which has electrons orbiting around it.

Every element on the periodic table is made up of the same type of atom.

An isotope of an element is an atom that has more (or fewer) neutrons than usual, but the same number of protons.

For example, deuterium (an isotope of hydrogen) has one neutron while hydrogen has none.
Another isotope of hydrogen is tritium, which has two neutrons.
